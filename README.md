## 📖 About The Project

This project is built for learning and practicing backend development using **Node.js** and **Express.js**.  
It covers the fundamentals of server-side programming, APIs, routing, middleware, database connection, and CRUD operations.

The project is suitable for beginners who want to understand how modern backend applications work.

---

# ✨ Features

✅ REST API Development  
✅ Express Server Setup  
✅ Routing System  
✅ Middleware Usage  
✅ MongoDB Database Connection  
✅ CRUD Operations  
✅ Error Handling  
✅ Environment Variables  
✅ JSON Data Handling  
✅ API Testing with Postman
